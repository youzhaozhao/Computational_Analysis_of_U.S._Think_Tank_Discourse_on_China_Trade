# Computational Analysis of U.S. Think Tank Discourse on China Trade (1980-2001)

### 话语即权力：1980-2001年美国智库对华贸易话语的商业化转型——基于计算史学与文本挖掘的实证研究


摘要：

从1980年美国国会首次批准给予中国最惠国待遇到2000年对华永久正常贸易关系法案通过，美国对华贸易政策经历了由遏制到接触的根本性转型。既有研究常将这一转型归因为地缘战略调整或利益集团资源竞争，较少关注观念层面的合法性建构。本研究采用计算史学的研究方法，利用文本挖掘与自然语言处理技术，构建了包含5951个文本片段的历史语料库，在工商界语料上微调GPT-2语言模型并计算美国主要智库对华政策文本的困惑度变化。结果显示，智库话语与商业话语在90年代中期后出现显著趋同，该趋势与企业政治献金强度呈明显负相关，且资金投入对智库话语影响有约1年的滞后性。微观文本对比分析进一步显示智库的话语模式在对华政策中发生明显偏移。本研究表明，资本不仅通过直接游说影响政策，也通过长期资助重塑智库等知识生产机构的话语，使商业利益逐步被包装为国家利益，从而为亲商政策提供认知合法性，揭示“金钱塑造话语、话语重构合法性、合法性支撑政策”传导机制的可能。
